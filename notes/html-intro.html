Let’s talk about the web

The web was created by Tim Bernes-Lee while working at CERN, the research institute in Geneva Switzerland. He wanted a way to lnk the many files on his collegues computers together. This action of linking pages together is called “Hypertext” and the language used to describe these pages is called HyperText Markup Language. The web is a collection of webpages.

Let’s talk about Browsers

Broswers are software used to view and explore webpages. You write HTML to describe to your browser what your web page should look like, what it should do and other details about your page. HTML files are saved with the .html extension.

HTML Syntax

The Syntax of a language are the rules guide the forming of statements in that language. In pidgin for example you don’t say dey I when someone asks how you are. Instead you respond with I dey. Same way HTML has rules that define how the language is written. Every HTML code is made up of elements, these are the building blocks of your webpage. You add elements to a page and can add embed elements also. Elements are represented by tags: 

<element name> content </element name>

An opening and closing tag with content in the middle. If you want to add a paragraph element, you put a paragraph tag like so: 

<p>content</p>

When the browser sees this tag, it will display a paragraph element with that text. You don’t have to memorize these tags, it’s how you use them that matter.

HTML Element Attributes

Most HTML elements have attributes you can use to change something about that element. You add attributes to the first opening tag of the element. Let’s say there was an element called “Student” (there is none btw). Without attributes our student element is a generic fellow. But when we add some attributes they get interesting! 

<student> I am a student </student>

Meh, Generic Student element

 <student name=”Emmanuel” level=”Senior” hotel=”Trivago”> I am a student </student>

Whoa! This is an interesting fellow!

HTML Webpage structure

Not all HTML elements/tags are for displaying something. Some provide information about the page and provide structure for other elements or the entire page.
This is the basic structure of a web page in HTML. You can see the <html> tag? That’s the element telling the browser that this is a HTML page. Then the <head> tags provide more data about our page and the <body> tag is where all our visual elements go. 

<html>

  <head> Nothing here will be displayed in the browser </head>

  <body> Everything here will be displayed in the browser </body>

</html>

Let’s make our first web page

To make a webpage we need to create a HTML file, for that we need a text editor. One common one that comes with all Windows computers is Notepad. On Mac, that will be TextEdit. Open your text editor and write out a basci webpage using the structure I outlined above. Then add some elements, like a Header tag <h1>Hello world!</h1>. You can find more about HTML elements on W3Schools. Generally, W3Schhols is a great resource for learning HTML, CSS and Javascript. Bookmark it ;)

Remember to place your visual elements (paragraphs, headers) inside the <body></body> tag. And don’t forget to close all your tags. When you are done, save your TXT file as a .html (or .htm) file and open it with your browser. You should see your page!
